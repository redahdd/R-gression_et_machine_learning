# R-gression_et_machine_learning
Ce dépôt regroupe le travail réalisé dans le cadre d’un TP de Régression et Machine Learning, dont l’objectif est d’approfondir l’étude de la régression linéaire, initialement abordée en travaux dirigés de statistiques, tout en mettant en évidence ses liens avec le domaine du machine learning.
